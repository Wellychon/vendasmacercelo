### 🚀 **Como Fazer Deploy**

#### **Opção 1: Deploy Automático**
```bash
./deploy-vercel-optimized.sh
```

#### **Opção 2: Deploy Manual**
```bash
# 1. Instalar Vercel CLI
npm install -g vercel

# 2. Login no Vercel
vercel login

# 3. Deploy
vercel --prod
```
